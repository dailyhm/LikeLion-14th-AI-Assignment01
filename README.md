머신러닝 활용

1.선택한 서비스
-개인신용평가

2.이 서비스는 무엇을 예측하는가
-개인의 신용 등급을 예측해준다

3.어떤 데이터를 사용하는가 (최소 3가지)
-대출 이력 및 상환 내역, 직장의 유무 및 소득 수준,지출 수준

4.이 모델은 어떻게 학습할까?
- 데이터들을 이용하여 신용 등급을 판단하는 기준에 따라 지도 학습을 하여 신용등급을 예측함

5.이 모델이 더 좋아지려면 어떻게 해야하는가?
- 해당 모델이 좋아지기 위해 3번에서 언급한 외의 데이터들을 넣거나 실제 사람이 매긴 신용 등급으로 비교를 해보며 게산의 정확성을 높혀서 신용등급을 더 정확하게 측정하면 될 것같다
